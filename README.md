# 燃气轮机全产业链图谱

本仓库用于保存和发布“燃气轮机全产业链图谱”网页，以及后续查询所需的项目上下文、版本记录、素材说明和部署信息。

## 在线访问

- GitHub Pages 首页：https://rhea0419.github.io/gas-turbine-chain-map/
- 具体页面：https://rhea0419.github.io/gas-turbine-chain-map/gas-turbine-chain-map.html

## 仓库结构

- `gas-turbine-chain-map.html`：当前线上页面。
- `assets/gas-turbine/`：页面实际引用的燃气轮机主图和产业链分区图片素材。
- `versions/`：重要历史版本备份。
- `docs/`：项目上下文、需求记录、数据及素材来源说明、后续版本管理建议。
- `references/`：预留目录，用于后续放置可公开或已脱敏的参考资料。
- `index.html`：GitHub Pages 首页跳转。

## 当前发布版本

- 日期：2026-07-10
- 版本说明：A4 双页排版，第一页为燃气轮机定义、工作原理、核心结构；第二页为产业链结构，包含上游材料、中游设备及制造、下游应用三大部分。
- 部署方式：GitHub Pages，从 `main` 分支根目录发布。

## 注意事项

当前仓库为公开仓库。为避免公开内部资料，本仓库目前不直接上传原始 Word 报告、调研材料或其他可能包含敏感信息的文件。相关文件名称、用途和本地路径在 `docs/source-inventory.md` 中记录，便于后续在本机或私有仓库中追溯。

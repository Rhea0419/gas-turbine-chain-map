# 项目上下文记录

## 项目目标

建设一个可视化、可追溯、可持续迭代的电力能源产业链图谱。当前 GitHub 仓库聚焦燃气轮机全产业链图谱页面，用于长期访问、版本管理和后续查询。

## 当前页面目标

燃气轮机页面用于以 A4 规格展示燃机科普和产业链结构，便于网页浏览、手机访问和导出为 PDF 图片。

## 已确定的燃气轮机页面设计要求

- 标题为“燃气轮机全产业链图谱”，居中放在页面顶部。
- 页面采用白色背景，整体风格统一、协调、适合导出。
- 第一页包含：
  - 核心定义
  - 工作原理
  - 核心结构
- 第一页主图使用重新绘制的燃气轮机剖面示意图，保留空气流入、高温燃气流动、能量输出等箭头关系。
- 主图下方展示文字说明：
  - 燃气轮机被誉为装备制造业“皇冠上的明珠”，是以连续流动气体为工质的高端动力装备，产业链覆盖特种材料、精密制造、能源、军工等核心领域，2025年全球市场规模约226亿美元。
- 第二页标题为“产业链结构”。
- 第二页包含：
  - 四 上游材料
  - 五 中游设备及制造
  - 六 下游应用
- 第二页需要完整展示图谱中的所有图片和文字信息，不采用点击切换隐藏内容。
- 第二页小图需要保持合理比例，不能被压扁或溢出边框。

## 重要交互和版本决策

- 当前燃气轮机页面优先满足“静态导出”和“PDF 图片化”需求，因此没有采用交互式折叠或点击切换。
- GitHub Pages 用于替代 Cloudflare 临时隧道，提供长期外网访问。
- 当前仓库公开发布，内部报告和原始资料不直接上传，仅记录索引。

## 后续建议

- 如果需要保存全部原始资料，建议新建一个私有仓库或将当前仓库转为私有。
- 如果需要同时发布知识图谱交互版和 A4 报告版，可在仓库中新增：
  - `interactive/`
  - `print/`
  - `data/`
  - `docs/decision-records/`
